Linux:

# Python 3

Install Python 3 in your system.

# Install

The script below will create a virtual env and install all required packages. Check the file in case of erros.

$ ./before_first_run.sh

Note: This process will fail without python3 in system

# Running

1. Exec run.sh to start your webserver
2. Open your browser
3. Open localhost:5000 or 127.0.0.1:5000
